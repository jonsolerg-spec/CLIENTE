# EliteWalk ERP

EliteWalk ERP es un sistema de planificación de recursos empresariales (ERP)
diseñado para apoyar la gestión interna de la empresa EliteWalk.

##  Estructura del proyecto

- /docs  
  Documentación general del proyecto.

- /design  
  Diseños de interfaz y prototipos del sistema.

- /diagrams  
  Diagramas UML, casos de uso y arquitectura.

- /src  
  Código fuente del sistema.

- /tests  
  Pruebas del sistema.

##  Enlaces importantes

- Documentación: /docs
- Diseño: /design
- Diagramas: /diagrams
- Código fuente: /src
- Pruebas: /tests
